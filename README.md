# mcping

ping checker for minecraft server like [paping](https://code.google.com/archive/p/paping/)

## usage

```python
mcping.py <ip> <pe or pc> (port)
```

## example 

```python
mcping.py 127.0.0.1 pc 
```
```python
mcping.py 127.0.0.1 pe
```
```python
mcping.py 127.0.0.1 pc 23231
```
